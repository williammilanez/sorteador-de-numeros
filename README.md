## 📚 Sobre o projeto

O **Sorteador de Números** é uma aplicação web desenvolvida para gerar números aleatórios dentro de um intervalo definido pelo usuário.

A aplicação permite:

- Definir quantidade de números a serem sorteados
- Definir valor mínimo e máximo do intervalo
- Sortear números com repetição
- Sortear números sem repetição
- Refazer sorteios mantendo os mesmos parâmetros
- Visualizar resultados de forma clara e responsiva

O projeto foi desenvolvido com foco em:

- JavaScript moderno (ES Modules)
- Separação de responsabilidades
- Manipulação do DOM
- Validação de dados
- Responsividade
- Boas práticas de desenvolvimento
